## üéØ Before Asking for Help

To help us help you effectively:

1. **Search First**: Check if your question has been answered
2. **Be Specific**: Provide clear descriptions and error messages
3. **Include Context**: Share your environment details and configuration
4. **Show Your Work**: Explain what you've already tried
5. **Use Markdown**: Format your code and logs for readability

### Good Question Example
```markdown
**Environment:**
- OS: Ubuntu 22.04
- Node.js: 18.17.0
- Project Version: 2.1.0

**Problem:**
Getting `TypeError: Cannot read property 'x' of undefined` when calling `processData()`

**What I've Tried:**
1. Checked that data is being passed correctly
2. Verified API response format
3. Added console.logs to debug

**Code Sample:**
\`\`\`javascript
const result = processData(input);
// Error occurs here
\`\`\`

**Full Error:**
\`\`\`
TypeError: Cannot read property 'x' of undefined
    at processData (index.js:42:15)
    at main (index.js:10:20)
\`\`\`
```
